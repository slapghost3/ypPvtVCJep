# 图书馆管理系统 python716

## 项目概述

本项目是基于 PythonGUI 的图书馆管理系统，旨在帮助图书馆高效管理图书资源，优化图书采购、淘汰、租借流程。

## 技术栈

- Python
- PyQt5（图形用户界面）
- MySQL（数据库）

## 功能模块

### 用户管理
- 用户注册
- 用户登录
- 修改密码

### 图书管理
- 存储图书信息
- 采购和淘汰情况管理
- 租借情况管理

### 功能实现
- 图书采购
- 图书淘汰
- 图书租借
- 图书信息查询
- 采购和淘汰查询
- 库存查询
- 租借情况统计

## 系统角色
- 系统管理员：负责用户管理和系统设置
- 图书管理员：负责图书采购、淘汰、信息管理等
- 读者：可查询图书信息，进行图书租借

## 运行环境
- 数据库环境：MySQL
- 操作系统：Windows（打包好的 EXE 可执行程序）

## 部署步骤
1. 配置好 MySQL 数据库环境
2. 双击 EXE 可执行程序直接运行
3. 系统自动引导至登录界面
